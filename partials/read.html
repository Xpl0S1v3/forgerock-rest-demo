To retrieve a single resource, perform an HTTP GET and accept a JSON response. 

GET /users/1  HTTP/1.1
Host: example.com
Accept: application/json

The following example retrieves Barbara Jensen's resource using the curl command. 

$ curl --request GET \
 --header "Accept: application/json" \
