About Deleting

To delete a single resource, perform an HTTP DELETE and accept a JSON response. 

DELETE /users/bjensen  HTTP/1.1
Host: example.com
Accept: application/json

You can use _prettyPrint=true to pretty print the JSON returned when working on the command line.
